# 🛡️ SOC Log Analyzer (Python Project)

## 📌 Project Overview
This project is a simple Security Operations Center (SOC) log analyzer built using Python. It reads security logs, extracts important information, and generates a report of detected threats.

It simulates basic SIEM-style log processing used in cybersecurity environments.

---

## ⚙️ Features

- Reads security logs from a file
- Detects and counts different security alerts
- Extracts attacker IP addresses
- Counts most frequent attackers
- Generates a structured security report (`report.txt`)
